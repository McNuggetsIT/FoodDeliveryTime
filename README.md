## Descrizione del Progetto
FoodDeliveryTime è un progetto di analisi dei dati e machine learning dedicato allo studio dei tempi di consegna nel settore del food delivery.  
L’obiettivo è individuare le variabili che influenzano maggiormente le tempistiche e costruire un modello predittivo accurato.

Il progetto comprende:
- Analisi esplorativa dei dati
- Pulizia e preparazione del dataset
- Feature engineering e selezione delle variabili
- Addestramento di modelli di regressione
- Valutazione e ottimizzazione delle performance

## Obiettivi di Data Analysis

### 1. Pulizia dei dati
- Gestione dei valori mancanti
- Correzione dei formati errati o non coerenti
- Conversione delle variabili categoriche in valori numerici

### 2. Normalizzazione dei dati
- Normalizzazione delle variabili numeriche
- Preparazione dei dati per i modelli di machine learning

### 3. Analisi Esplorativa e Visualizzazioni
- Analisi delle distribuzioni
- Identificazione di outlier
- Creazione di grafici descrittivi
- Analisi del tempo di consegna in funzione delle principali feature

### 4. Feature Engineering
- Creazione di nuove variabili rilevanti
- Imputazione e normalizzazione del Traffic Level tramite algoritmo
- Trasformazione dei mezzi di trasporto tramite get_dummies

### 5. Correlazioni
- Analisi della matrice di correlazione
- Identificazione delle feature più influenti sulle tempistiche

## Obiettivi di Machine Learning

### 1. Pre-processing Avanzato
- Imputazione e normalizzazione del Traffic Level
- Encoding delle categorie dei mezzi di trasporto
- Riduzione della dimensionalità per migliorare l’efficienza dei calcoli

### 2. Modellazione
- Addestramento di un modello di regressione per prevedere il tempo di consegna
- Valutazione di modelli alternativi (es. Random Forest, Gradient Boosting)
- Calcolo di metriche come MAE, RMSE, R²

### 3. Ottimizzazione
- Ricerca degli iperparametri migliori
- Analisi degli errori del modello
- Miglioramento della generalizzazione

## Risultati Attesi
- Identificazione delle variabili più rilevanti che influenzano il tempo di consegna
- Sviluppo di un modello predittivo affidabile
- Possibile utilizzo per ottimizzare le operazioni di un servizio di food delivery

## Conclusioni
Il progetto combina analisi statistica, trasformazione dei dati e machine learning per sviluppare un sistema di previsione dei tempi di consegna.  
L’obiettivo finale è comprendere meglio le dinamiche operative e migliorare l’efficienza delle consegne nel food delivery.
